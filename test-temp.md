# 类型推导基础
## 为什么需要类型推导
在很多编程语言中，你需要为每一个变量显式标注类型：
```
    let x: i32 = 5;
    let name: String = String::from("Alice");
    let nums: Vec<i32> = Vec::new();
```
这样做很冗长。Rust 设计了一个**聪明的类型推导引擎**，让编译器自动推断变量的类型。
## 基本推导规则
### 从初始化值推导
最直接的方式是从右边赋予的值推导类型：
```
    fn main() {
        let x = 5;
        let y = 5.0;
        println!("x: {}, y: {}", x, y);
    }
```
> **提示**：类型推导是 Rust 的核心特性之一。
## 总结
类型推导让 Rust 代码简洁又安全。
